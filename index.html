Sobre o Autoconnectx: O Autoconnectx visa facilitar para todos os publicos a compra de peças de carros. Diminuindo a depedencia em mecânicos de indole suspeita. O diferencial do aplitivo é que você sabendo o ano e modelo do seu carro, o aplicativo já oferece a peça disponivel pro local danificado.

Como uasr o Backend. 1- Git clone 2- Npm Install 3 - Npm Start

Tecnologias usadas: JavaScript Node.js Express.js MongoDB Mongoose Nodemon Axios

Rotas http://localhost:5000/() - Clientes - Vendedores - Pecas - Carrinho - Item carrinho - Pedidos
